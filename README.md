### Â¿Que es?

La contraccion E.N.N.I. significa "El nombre no importa" y mas que un titulo es una filosofia de desarrollo en la que no
se invierte tiempo o recursos en elementos que no benefician directamente la calidad del proyecto.

Es un entorno integrado para la simulacion de Robots utilizando arduino como plataforma principal similar a LEGO Mindstorms
pero con la caracteristica que permite la simulacion de propiedades fisicas.

El IDE se comunica con la plataforma Arduino por medio del Firmware Firmata y proporciona una interaccion por comunicacion 
serial en tiempo real.
