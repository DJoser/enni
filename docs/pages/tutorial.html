E.N.N.I.

ENNI es un simulador por computadora el cual puede ser conectado a robots reales para ser manipulados de forma virtual y real, es facil de extender por medio de plugins atravez de python.

El proyecto es diseñado y construido por Jose Ramon Navarro Marquez, Carlos Muños Villareal y Alfredo Pizaña y fue construido con la finalidad de servir como proyecto de titulacion para la carrera de Ingenieria en desarrollo de software en el Centro de Enseñanza Tecnica industrial.

Objetivos

Dentro de los objetivos generales del proyecto se busca realizar una implementacion utilizando las tecnologias mas actuales de una manera organizada, en el proyecto se convinan la potencia de C++ con Opengl para la generacion de graficos, La versatilidad de Html/Javascrit/Css para el diseño de la interfaz grafica con la que interactua el usuario, y la amplitud del proyecto Python el cual sirve como la conexion con interfaces externas.
